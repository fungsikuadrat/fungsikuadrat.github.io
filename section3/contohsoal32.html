  1. Perhatikan grafik fungsi kuadrat pada gambar berikut ini.
    Tentukan persamaan fungsi kuadrat memalui titik potong$-x$ dari grafik tersebut.
    Pembahasan :
    Diketahui : Titik potong sumbu$-x = (1, 0) (5, 0)$
    Titik potong sumbu$-y=(0,-5)$
    Ditanyakan : Persamaan fungsi kuadrat?
    Jawab : $y=a(x-x_1)(x-x_2)$
    $: y=a(x-1)(x-5)$
    $: y=a(0-1)(0-5)$
    $: -5=a5$
    $: a=-\frac{5}{5}=-1$
    Sehingga diperoleh persamaan fungsi kuadrat :
    $=a(x-x_1)(x-x_2)$
    $=-1(x-1)(x-5)$
    $=-(x^2-5x-x+5)$
    $=-(x^2-6x+5)$
    $=-x^2+6x-5$
  1. Perhatikan grafik fungsi kuadrat pada gambar berikut ini.

    Tentukan persamaan fungsi kuadrat memalui titik potong$-x$ dari grafik tersebut.
    Pembahasan :
    Diketahui : Titik potong sumbu$-x = (-1, 0) (3, 0)$
    Titik potong sumbu$-y=(0,-6)$
    Ditanyakan : Persamaan fungsi kuadrat?
    Jawab : $y=a(x-x_1)(x-x_2)$
    $: y=a(x+1)(x-3)$
    $: y=a(0+1)(0-3)$
    $: -6=a(-3)$
    $: 3a=6$
    $: a=-\frac{6}{3}=2$
    Sehingga diperoleh persamaan fungsi kuadrat :
    $=a(x-x_1)(x-x_2)$
    $=2(x-1)(x-5)$
    $=2(x^2-3x+x-3)$
    $=2(x^2-2x-3)$
    $=2x^2-4x-5$
